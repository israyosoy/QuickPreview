```
QuickPreview brings the beloved Quick Look experience to Windows. Select any file in File Explorer and press Space for an instant preview — no need to open any application.

🖼️ Images
JPG, PNG, GIF, BMP, WEBP, TIFF, ICO, SVG

• Excel (XLSX, XLS) — switch between sheets, no macros

• JavaScript fully disabled in the document viewer
• No macro or formula execution
• SVG automatically sanitized
• No internet connection — all previews are local and private
```

## NOTAS PARA LA REVISIÓN DE MICROSOFT

En el campo "Additional information" / "Notas para el revisor" de Partner Center, incluye:

```
Esta aplicación utiliza SetWindowsHookEx (WH_KEYBOARD_LL) para interceptar
la tecla Espacio globalmente y activar previsualizaciones de archivos cuando
el Explorador de Windows está activo. Este comportamiento es la función
principal de la app y está documentado en la descripción. La app no
transmite datos de pulsaciones de teclas a ningún servicio externo.
```

**Título:** Política de Privacidad de QuickPreview

QuickPreview no recopila, almacena ni transmite ningún dato personal ni información sobre el uso de la aplicación.

La aplicación funciona completamente de forma local en el equipo del usuario. No realiza conexiones a internet, no envía telemetría y no muestra publicidad.

Los archivos que se previewan no se cargan en ningún servidor externo — todo el procesamiento ocurre en el dispositivo del usuario.

Esta política fue actualizada por última vez en junio de 2026.

---
*Privacy Policy — English version:*

QuickPreview does not collect, store, or transmit any personal data or usage information.

The application operates entirely locally on the user's device. It makes no internet connections, sends no telemetry, and shows no advertising.

Files previewed are not uploaded to any external server — all processing occurs on the user's device.
